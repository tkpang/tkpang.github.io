# 世界历史时间线 - 使用说明

## 简介

这是一个交互式的世界历史文明时间线可视化工具，展示了从公元前3500年到现代的主要文明发展历程。

## 功能特点

### 1. 交互式可视化
- **缩放**：使用鼠标滚轮可以放大或缩小时间线
- **拖动**：点击并拖动画布可以移动视图
- **点击**：点击任何文明方块可以查看详细信息

### 2. 时间范围筛选
- 全部历史 (5000年)
- 古代 (公元前3000-公元500)
- 中世纪 (500-1500)
- 近现代 (1500-2000)

### 3. 地区筛选
- 全部地区
- 亚洲（东亚、南亚、西亚）
- 欧洲
- 非洲
- 美洲

### 4. 详细信息面板
点击任何文明方块后，会弹出详情面板显示：
- 文明名称
- 时期（起始和结束年份）
- 所属地区
- 文明简介
- 重大历史事件

## 访问方式

### 本地测试
1. 启动Jekyll服务器：
   ```bash
   bundle exec jekyll serve
   ```

2. 在浏览器中访问：
   ```
   http://localhost:4000/history-timeline.html
   ```

### GitHub Pages
1. 提交并推送所有文件到GitHub
2. 访问：

## 如何添加新的文明

编辑 `js/history-data.js` 文件，在对应地区的数组中添加新条目：

```javascript
{
    name: "文明名称",
    region: "地区代码", // eastAsia, southAsia, westAsia, europe, africa, americas
    startYear: -1000,   // 开始年份（负数表示公元前）
    endYear: 500,       // 结束年份
    color: "#e74c3c",   // 颜色代码
    description: "文明的简短描述",
    events: [
        "重要事件1",
        "重要事件2",
        "重要事件3"
    ]
}
```
